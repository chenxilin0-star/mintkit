# MintKit

> AI-powered digital product generator — turn any niche into sellable PDF products in minutes.

## What It Does

1. **Enter a niche** (e.g., "dog training", "french learning for kids")
2. **Get 5 product ideas** generated by GPT-4o (Planner / Checklist / Guide)
3. **Select one** → AI generates the complete product content
4. **Download PDF** and upload to Gumroad to start selling

## Tech Stack

- **Framework:** Next.js 14 (App Router) + TypeScript
- **Styling:** Tailwind CSS
- **AI:** GPT-4o (OpenAI API)
- **PDF:** html2pdf.js (client-side rendering)
- **Deploy:** Vercel

## Getting Started

### 1. Clone & Install

```bash
git clone https://github.com/chenxilin0-star/mintkit.git
cd mintkit
npm install
```

### 2. Add OpenAI API Key

Create `.env.local` in the project root:

```env
OPENAI_API_KEY=sk-your-key-here
```

### 3. Run Dev Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### 4. Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Set the `OPENAI_API_KEY` environment variable in Vercel dashboard.
